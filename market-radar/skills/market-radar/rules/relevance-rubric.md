# Relevance rubric (lite principles)

Edit this for your business. The radar keeps an item only when **both** tests pass:

## Keep test

1. **Assumption link** — Which watched assumption does this touch? If none, drop.
2. **Principle link** — Which decision rule makes it matter? If none, drop.
3. **Decision window** — Could you act on this in the next 2–4 weeks? If not, drop or mark `watch_only`.

## Stance

| Stance | When |
|---|---|
| `threat` | Breaks or activates a fragile assumption negatively |
| `opportunity` | Supports a bet you're making or opens a move |
| `mixed` | Helps one assumption, hurts another |
| `watch_only` | Real signal, no immediate move |

## Relation (to assumption)

`supports` · `contradicts` · `activates` · `inhibits` · `context_only`

## Recommended action

`brief` · `investigate` · `simulate` · `snooze` · `ignore`

Always include **evidence** (what the source says), **warrant** (why it bears on the assumption),
and preferably **rebuttal** (what would weaken the read).
